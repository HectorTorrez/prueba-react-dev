# React Prueba Tecnica

## Demo

https://prueba-react-dev.vercel.app/

## Environment Variables

Para poder correr este proyecto necesitaras agregar estas variables de entornor a tu .env.local

`API_KEY`

`APP_ID`

`NEXT_PUBLIC_API_KEY_TOMTOM`

## Correr localmente el proyecto

Clona el repositorio

```bash
  git clone https://github.com/HectorTorrez/prueba-react-dev
```

Ve al proyecto

```bash
  cd my-project
```

Instala las dependencias

```bash
  yarn install
```

Inicia el proyecto

```bash
  yarn dev
```
